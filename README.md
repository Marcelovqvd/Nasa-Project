# Nasa Project


## API
https://api.nasa.gov/


## Para iniciar o projeto

Instale o Node.js;

CLone o repositório:

    $ git clone https://github.com/Marcelovqvd/Nasa-Project.git

Instale as dependências do projeto:

    $ yarn install

Execute a aplicação:

    $ yarn start
